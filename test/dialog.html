alert
使用时需要引入eui.dialog.min.js。

通过调用 eui.alert() 来模拟原生 alert() 的效果。使用简单，效果美观。

不过从用户体验角度来说，应该尽可能避免这类打断用户的交互方式。

通过调用 eui.confirm() 来模拟原生 confirm() 的效果。

通过调用 eui.info() 来显示提示对话框（点击对话框外可隐藏对话框）

方法
方法 说明
eui.alert(options) 功能类似 Javascript 的 alert。
eui.alert('msg') 显示一段文本。
eui.confirm(options) 功能类似 Javascript 的 confirm。
eui.confirm('msg') 显示一段文本。
参数介绍
选项 默认值 说明
title string 对话框标题
msg string 对话框内容
