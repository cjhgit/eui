eUI 后台主题UI框架

eUI 是一个完全响应式，基于 Bootstrap3.3.6 最新版本开发的扁平化主题，借（chao）鉴（xi）了许多优秀的 UI 框架。

本着不重复造轮子的原则，只对 Bootstrap 进行了少量的修改。

我们不生产代码，我们只是代码的搬运工。

当前版本：v4.0.0

框架下载 作者博客

主题特性

  1. 完全响应式布局（支持电脑、平板、手机等所有主流设备）
  2. 基于最新版本的Bootstrap 3.3.4
  3. 提供3套不同风格的皮肤
  4. 支持多种布局方式
  5. 修改了部分样式（不是 Bootstrap 不好看，而是防止用户审美疲劳）
  6. 使用最流行的的扁平化设计
  7. 提供了诸多的UI组件
  8. 集成多款国内优秀插件，诚意奉献
  9. 提供盒型、全宽、响应式视图模式
  10. 采用HTML5 & CSS3
  11. 拥有良好的代码结构
  12. 更多……

兼容性

IE 8+ 等主流浏览器

考虑到国内用户，eUI 基于 Bootstrap 3 而不是 Bootstrap 4 开发。
