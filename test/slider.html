方法
方法 说明
$(selector).silder(options) 初始化轮播为可选的 options 对象，并开始循环项目。
$(selector).silder('cycle') 从左到右循环轮播项目。
$(selector).silder('pause') 停止轮播循环项目。
$(selector).silder(number) 循环轮播到某个特定的帧（从 0 开始计数，与数组类似）。
$(selector).silder('prev') 循环轮播到上一个项目。
$(selector).silder('next') 循环轮播到下一个项目。
选项
选项 类型 默认值 说明
interval number 5000 自动循环每个项目之间延迟的时间（毫秒）。如果为 false，轮播将不会自动循环。
pause string hover 鼠标进入时暂停轮播循环，鼠标离开时恢复轮播循环。
wrap boolean true 轮播是否连续循环。
事件
事件 说明
slide.eui.slider 当调用 slide 实例方法时立即触发该事件。
slid.eui.slider 当轮播完成幻灯片过渡效果时触发该事件。
